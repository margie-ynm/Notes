Atom Pair Setup - (Open Terminal) - touch .pairs - atom .pairs (create pairs and email sections) Git Guide - Git init (starts tracking directory) - Git pair jr jp (adds authors to the git project) - Git add . (adds all files to watchlist) - Git status (to confirm directory is watched) - Git-pair-commit -m “Comment goes here” (to commit as a pair) - (Create Repository) - Git remote add jp url.com/ (Linking to repository, per person) - Git push jr master (pushes files to github, per person) - Git status (to confirm everything is up to date) Git Branching - git branch gh-pages (gh-pages can be any name) - git checkout gh-pages (switches to branch) - git push jp gh-pages (per person, pushes to branch) - git branch (displays branches, star denotes current branch)
